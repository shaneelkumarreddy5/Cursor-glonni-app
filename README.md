Modern web application scaffold for an e-commerce platform called **Glonni**.

## What is included

- React + TypeScript + Vite setup
- Responsive reusable layout shell
- Clean client-side routing with nested settings routes
- Placeholder UI pages (no backend/business logic)

## Pages

- Home (Landing)
- Category (PLP)
- Product (PDP)
- Cart
- Checkout
- Settings
  - Orders
  - Wallet
  - Support
  - Profile
  - Addresses
  - Notifications

## Run in development mode (live preview / HMR)

```bash
npm install
npm run dev
```

The dev server is configured to bind to `0.0.0.0` for live preview access.
